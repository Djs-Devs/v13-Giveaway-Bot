# v13-Giveaway-Bot
It's An Advance Giveaway Bot With Slash

# Requirements
```js
Discord.js ~ v13.6.0
```

# Fill Config.json Properly

```js
{
  "token": "Put your Token Here", 
  "prefix": "Put your prefix here",
  "everyoneMention": false,
  "hostedBy": true
}
```

# Start

- npm i
- npm start
